# Groundwater Monitoring System

## Project Structure
- `backend-node/`: Node.js backend
- `frontend/`: React frontend

## Setup & Installation

The project dependencies have been removed to save space. To restore them and run the application:

1.  **Install Backend Dependencies**:
    ```bash
    cd backend-node
    npm install
    ```

2.  **Install Frontend Dependencies**:
    ```bash
    cd frontend
    npm install
    ```

## Running the Application

1.  **Start Backend**:
    ```bash
    cd backend-node
    npm start
    ```

2.  **Start Frontend**:
    ```bash
    cd frontend
    npm start
    ```
    (Or `node node_modules/react-scripts/bin/react-scripts.js start` if `npm start` fails).

## Cleaning the Project

To reduce the project size for sharing/zipping, run the cleanup script from the root directory:

```bash
./clean_project.sh
```
